# ML Scripts

This folder contains offline machine-learning scripts:

- `train.py`: model training
- `test.py`: offline evaluation
- `preprocess.py`: dataset preprocessing

Backward-compatible root entrypoints are kept:

- `python train.py`
- `python test.py`
- `python preprocess.py`

You can also run as modules:

- `python -m ml.train`
- `python -m ml.test`
- `python -m ml.preprocess`
